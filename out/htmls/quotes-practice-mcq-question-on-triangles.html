1.
If ABC and PQR are similar triangles in which ∠ A = 470 and ∠ Q = 830, then ∠ C is:


Answer & Solution Discuss in Board Save for Later

Answer & Solution

Answer: Option A
Solution: Since, Δ ABC and Δ PQR are similar triangles.
then,∠ B = ∠ Q = 830
Thus, in Δ ABC,
∠ C = 1800 - (∠ A + ∠ B)
or, ∠ C = 1800 - (470 + 830)
∠ C = 500.
